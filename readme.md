### Setup del ambiente de desarrollo
#### Requisitos
- Docker y Docker Compose

#### Instrucciones
1. ``git clone https://github.com/carlosmgc2003/juntosajugar.git``
2. ``cd juntosajugar``
3. ``docker-compose up``
